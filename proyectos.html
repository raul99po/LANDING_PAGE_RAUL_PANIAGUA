Gestión de bases de datos.



Esta bases de datos ha sido diseñada para gestionar eficientemente las operaciones de una escuela. Su objetivo principal es almacenar y organizar la información relacionada con los alumnado, grados y modulos donde están matriculados.
Es un trabajo de la asignatura BBDD. Uso código SQL. Este proyecto se realizó en la interfaz "MySQL Workbench 8.0 CE".
En este proyecto se crea tanto la base de datos, las tablas que la componen, sus atributos, relaciones entre distintas tablas mediante Foreign Key y valores dados al azar para comprobar el correcto funcionamiento de la misma.

SS.OO Linux. Script básico.



Este script de shellscript muestra un menú interactivo que permite a un administrador de sistemas operativos realizar una serie de tareas de administración.
Cada opción del menú realiza una tarea específica, como comprobar si un archivo de texto está vacío, verificar si dos números son distintos, mostrar la ruta del directorio actual, o salir del menú.
Es un trabajo de la asignatura ISO. Este script se ha programado en un entorno Ubuntu.

SS.OO Linux. Script NMAP.




Este script creado en shellscript analiza tu red para ver que direcciones IP están conectadas y analiza que puertos están abiertos en un dispositivo.
Cuando analizas el dispositivo, te muestra los puertos abiertos, las versiones e información adicional sobre la máquina víctima.

SS.OO LINUX. Script de monitorización de procesos.



Es script monitorea constantemente un proceso que el usuario especifica, revisa si está corriendo y, si no lo está, intenta reiniciarlo. Además, envía una alerta por correo electrónico al usuario cuando el proceso es reiniciado. Toda la actividad se registra en un archivo de log que también es definido por el usuario.
